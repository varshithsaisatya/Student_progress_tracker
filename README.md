# Student_progress_tracker
The Student Progress Report System is a Python-based to manage and evaluate a student’s academic performance. The program collects student details, records subject-wise marks, calculates total and average marks, assigns grades, determines pass/fail status, and provides a menu-driven interface for viewing results.

# Features 
•Accepts student details (Name, Class, Roll Number)

•Takes marks for multiple subjects

•Automatically calculates:

   •Total marks
  
   •Average marks
  
•Assigns grades based on average

•Determines pass/fail status

•Displays a detailed progress report

•Menu-driven system to view:

   •Total marks
  
   •Average marks
  
   •Full report
  
•Simple and user-friendly console output
